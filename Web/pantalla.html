ZeroKeyUSB Screens in SVG

Welcome Screen – Page 0

Displays "First Startup", "ZeroKey USB" and an indicator "---------->" at startup.

Welcome Screen – Page 1

Invites the user to check "About" for more information and support.

Key Creation Screen

Indicates that the encryption key is being generated.

Ready to Use Screen

Displays that the device is ready for use.

PIN Screen

Displays the entered PIN in large digits and a box on the right in white with a black number indicating the next digit to select.

Main Screen

Displays an indicator on the left edge and two lines of text. The first line shows a globe icon next to the site name (8 characters), and the second line shows a user icon next to the username (8 characters). The larger font forces each line to display only 8 characters.

Edit Screen

Displays the editable field and simulates keyboards and cursors to modify the values of Site, User, and Pass.

Menu Screen

Simulates the main menu display: shows the "MENU" indicator on the left edge and three options ("Backup", "Settings", and "Danger Zone"), with "Backup" highlighted.
